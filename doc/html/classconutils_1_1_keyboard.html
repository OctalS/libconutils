int conutils::Keyboard::waitForKey ( int  timeout_ms = -1)

Waits for a key to be pressed.

Parameters
timeout_ms: Timeout in milliseconds. -1 == wait indefinitely.
Returns
One of the following:
  • A valid ASCII printable key
  • One of the special keys. If alt, ctrl or shift is in combination it will be arithmetically added to the returned key. For example MOD_ALT + KEY_UP
  • -ETIMEDOUT if a time out was specified. Other negative codes are errors.
